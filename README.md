# calc-typescript
## Description
A simple calculator app built with React Native and TypeScript, featuring light/dark mode switching.

## Features
- Light/Dark Mode toggle
- Simple and easy-to-use UI for basic calculations

## Technologies
- React Native
- TypeScript
